PCO Start Date Generator

A piece of code that predicts the start date of a document

A Private Hire Driver's Licence (PCO) is a mandetory document that is required for drivers to hold if they wish to partner with companies like Uber, Bolt and Ola in London, UK.

At my previous employer (a competitor of the above mentioned companies) we found an opportunity where we could save time autocompleting the start date of a PCO if we already knew the expiry date.

By getting the driver to upload the document and input the expiry date only, we were able to predict the start date of the document automatically with the code below.

The introducton to this script meant we could remove the start date requirement for drivers when they were uploading documents, save time for document validation agents and increase accuarcy when auditing documents internally.

In this case, the example expiry date is '13/08/2022'.

The code takes three years away from 2022, then adds one day to 13.

The result = '14/08/2019'

Code:


  

import datetime

expiry_date = '13/08/2022'

exp_day = datetime.datetime.strptime(expiry_date, '%d/%m/%Y').strftime('%d')

month = datetime.datetime.strptime(expiry_date, '%d/%m/%Y').strftime('%m')

year = datetime.datetime.strptime(expiry_date, '%d/%m/%Y').strftime('%Y')

start_day = str(int(exp_day) + 1)

start_month = month

start_year = str(int(year) - 3)

start_date = start_day + "/" + start_month + "/" + start_year

print("Original expiry date: " + expiry_date)
print("Predicted start date: " + start_date)
print("The predicted start date has added one day to the 'day'
and deducted three from the 'year'")
